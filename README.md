Personal Website of Thanik Bhongbhibhat <email@thanik.me>

Site built with [Next.js](https://github.com/zeit/next.js).

## Developement

```bash
yarn start
```

## Production Build

```bash
yarn build-static
```

## Test Build Locally

```bash
yarn firebase serve  # after yarn build-static
```

## Deploy

```bash
yarn deploy  # after yarn build-static
```
